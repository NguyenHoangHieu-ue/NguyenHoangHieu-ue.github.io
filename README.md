## 📖 Quá trình thực hiện
Trang web này được mình xây dựng nhằm tối ưu hóa việc trưng bày năng lực cá nhân tới nhà tuyển dụng và giảng viên.
1. **Lên ý tưởng:** Xác định các khối nội dung cần thiết (Giới thiệu, Dự án, Chứng chỉ, Liên hệ).
2. **Lựa chọn công nghệ:** Sử dụng Template HTML5/CSS3 để đảm bảo giao diện chuyên nghiệp và hiển thị tốt trên mọi thiết bị.
3. **Chỉnh sửa & Tối ưu:** Sử dụng VS Code để tinh chỉnh mã nguồn, thay đổi thông tin cá nhân và cập nhật link các dự án thực tế từ GitHub.
4. **Triển khai:** Sử dụng GitHub Pages để host trang web trực tuyến một cách nhanh chóng và ổn định.
